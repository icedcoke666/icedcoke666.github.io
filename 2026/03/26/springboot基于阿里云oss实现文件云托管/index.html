今日知识点：springboot中如何实现文件的上传

📦 AliOSS 相关类的核心联系

  1. AliOssProperties：读取application.yml 文件中阿里云 OSS 配置信息
  2. OssConfiguration：基于AliOssProperties，阿里云文件上传配置类,用于创建AliOssUtil对象
  3. 封装Oss常用操作，实现文件上传
    这三个类是 阿里云 OSS 对象存储 在 Spring Boot 项目中的典型分层结构，共同完成「配置读取 → 工具封装 → 配置类注册」的完整流程。

调用关系与流程

1.AliOssProperties类

2.OssConfiguration类

3.AliOssUtil类

三者的协作逻辑

  • AliOssProperties 是「数据源」：负责把配置文件里的字符串变成 Java 对象。
  • OssConfiguration 是「工厂」：用配置对象创建出可直接使用的 OSS 客户端 Bean。
  • AliOssUtil 是「服务员」：基于 OSS 客户端封装易用方法，供业务层调用，避免重复写 OSS 原生 API。

一句话总结

AliOssProperties 读配置 → OssConfiguration 造客户端 → AliOssUtil 封装工具方法，三者共同实现了阿里云 OSS 在 Spring Boot 中的优雅集成。

CommonController：实现文件上传方法（有业务需要上传文件，直接调用即可）

注：properties类和util类放在common模块下，configuration类放在server模块下
